# SSH Manager - 功能更新说明

## 版本 1.1.0 - 重大更新

### 1. 交互式Shell会话 ✅

**改进内容：**
- 从单次命令执行升级为真正的交互式Shell会话
- 键盘输入实时发送到远端服务器并持续回显
- 快速命令复用同一个会话连接
- 支持Shell窗口大小调整（自动响应终端resize事件）
- 会话断开连接时自动提示

**实现细节：**
- 使用ssh2库的shell()方法创建真正的PTY会话
- 通过事件机制实时传递Shell数据到前端xterm.js
- 支持命令历史、自动补全（服务器端）

### 2. 终端分屏功能 ✅

**新增功能：**
- 支持左右水平分屏（⬌按钮）
- 支持上下垂直分屏（⬍按钮）
- 每个分屏可以独立运行不同的会话
- 可以将现有会话分配到任意分屏
- 关闭某个分屏不影响其他分屏和标签页
- 分屏后仍可继续使用标签页管理

**操作方式：**
1. 先创建一个会话
2. 点击"⬌"或"⬍"按钮进行分屏
3. 在新的空分屏中选择要显示的会话
4. 点击分屏右上角的"×"关闭该分屏

### 3. 项目分组管理 ✅

**完整功能：**
- 创建项目分组（支持自定义颜色）
- 重命名项目分组
- 删除项目分组（连接不会删除，仅移出分组）
- 在保存连接时选择所属项目
- 按项目筛选连接
- 项目卡片显示连接数量

**操作方式：**
1. 点击左侧"新建项目"按钮
2. 输入项目名称和选择颜色
3. 在创建/编辑连接时选择所属项目
4. 点击项目卡片直接筛选该项目的所有连接

### 4. 批量连接功能 ✅

**新增功能：**
- 连接列表支持多选（复选框）
- 支持全选/取消全选
- 批量打开多个终端会话
- 逐个连接，失败不影响其他连接
- 显示详细的连接结果报告：
  - 成功连接的服务器数量
  - 失败连接的服务器列表和失败原因
  - 清晰的失败原因提示

**操作方式：**
1. 在连接库中勾选多个要连接的服务器
2. 点击"批量连接"按钮
3. 系统会逐个尝试连接
4. 连接完成后显示详细的成功/失败报告

### 5. 文件浏览双窗口 ✅

**新增功能：**
- 左右双窗口布局，同时显示本地和远端文件
- 左侧：本地文件系统浏览
- 右侧：远程服务器文件浏览
- 支持选择本地目录
- 双向拖拽传输文件
- 上传/下载按钮快速操作
- 传完后自动刷新两边列表

**文件传输方式：**
1. **拖拽上传**：从本地面板拖拽文件到远程面板
2. **拖拽下载**：从远程面板拖拽文件到本地面板
3. **按钮上传**：选中本地文件后点击"上传 →"按钮
4. **按钮下载**：选中远程文件后点击"← 下载"按钮

**传输完成：**
- 自动刷新两边的文件列表
- 显示传输进度
- 失败时显示错误信息

## 已知限制

1. **分屏功能**：分屏后暂不支持动态调整分屏比例
2. **文件传输**：暂不支持大文件分片上传
3. **代理功能**：需要在设置中手动配置
4. **图标**：使用默认Electron图标，未自定义应用图标

## 使用建议

### 日常运维
1. 将常用服务器按项目分组
2. 使用收藏功能标记重要服务器
3. 使用批量连接快速打开多个会话

### 终端操作
1. 连接后即可直接输入命令（类似真实终端）
2. 使用快速命令面板执行常用操作
3. 可以拆分多个终端同时监控不同服务器

### 文件传输
1. 打开文件浏览器并连接服务器
2. 在左侧选择本地目录
3. 拖拽文件即可上传/下载
4. 查看两边列表确认传输结果
