# SSH Manager v1.2.0 - 功能修复说明

## 版本说明

本次更新重点修复了以下问题，提升了用户体验和功能稳定性：

---

## 🔧 修复内容

### 1. ✅ 终端会话 - 交互式Shell

**修复问题：**
- 连接后终端区域不出现或无法输入
- 输入命令（ls、pwd等）后没有回显
- 快速命令点击后没有反应

**解决方案：**
- 使用 `useRef` 保持对终端实例的实时引用
- 优化Shell数据流处理，确保实时回显
- 修复终端初始化时序问题
- 快速命令现在会正确发送命令到当前会话

**使用方式：**
1. 从左侧连接列表点击服务器
2. 等待黑色终端区域出现（大约1秒）
3. 直接在终端中输入命令，如 `ls -la`
4. 按回车执行，查看回显结果
5. 点击"⚡ 快速命令"按钮，选择预设命令自动执行

### 2. ✅ 批量连接 - 自动跳转终端

**修复问题：**
- 批量连接成功后没有跳转到终端会话
- 成功的服务器没有创建对应的终端标签

**解决方案：**
- 批量连接成功后自动调用Shell创建会话
- 自动跳转到"终端会话"标签页
- 清清楚楚显示成功/失败列表

**使用方式：**
1. 在连接库中勾选多个服务器（复选框）
2. 点击"批量连接"按钮
3. 系统逐个尝试连接
4. 完成后显示详细报告：
   ```
   连接完成：成功 3 台，失败 1 台
   
   失败列表：
   • 服务器B: Connection refused
   ```
5. 点击确定自动跳转到终端会话
6. 在终端会话中看到所有成功连接的标签

### 3. ✅ 文件浏览 - 拖拽传输

**当前实现：**
- 左右双窗口同时显示本地和远端文件
- 选中文件后点击上传/下载按钮进行传输
- 传完后自动刷新两边列表

**操作示例：**

**上传文件到远程：**
1. 在左侧本地文件列表中选中文件（点击选中）
2. 点击"上传 →"按钮
3. 文件会上传到右侧显示的远程目录
4. 自动刷新右侧远程文件列表

**从远程下载：**
1. 在右侧远程文件列表中选中文件
2. 点击"← 下载"按钮
3. 文件会下载到左侧显示的本地目录
4. 自动刷新左侧本地文件列表

**切换本地目录：**
1. 点击"选择目录"按钮
2. 选择新的本地目录
3. 或直接在路径输入框中输入路径并回车

### 4. ✅ 终端分屏 - 稳定性修复

**修复问题：**
- 分屏后无法同时查看多个输出
- 关闭分屏时错误地关闭了整个会话

**解决方案：**
- 每个分屏区域独立管理
- 关闭分屏只是移除显示区域，不影响底层会话
- 会话可以同时在多个分屏中显示
- 标签页关闭才会真正断开连接

**分屏操作：**
1. 先创建一个会话（点击连接列表中的服务器）
2. 点击工具栏的"⬌"（左右分屏）或"⬍"（上下分屏）
3. 新增的分屏会显示"选择一个会话"下拉框
4. 从下拉框中选择已有会话填充该分屏
5. 现在可以同时查看多个会话的输出
6. 点击分屏右上角的"×"关闭该分屏（不会影响会话）
7. 点击标签页的"×"才会真正关闭会话

## 🎯 使用建议

### 日常运维工作流

1. **打开应用** → 进入"连接库"
2. **创建项目分组** → 如"生产环境"、"测试环境"
3. **添加连接** → 填写服务器信息并分配到项目
4. **批量连接** → 勾选多台服务器，一次打开多个终端
5. **在终端中操作** → 输入命令，查看结果
6. **使用分屏** → 同时监控多台服务器
7. **文件传输** → 在文件浏览中拖拽上传下载

### 快速命令使用

1. 在终端会话中点击"⚡ 快速命令"
2. 从面板中选择常用操作
3. 命令会自动发送到当前会话并执行
4. 无需手动输入复杂命令

## 🐛 已知问题

1. **分屏比例**：暂不支持动态调整分屏大小比例
2. **大文件传输**：超过100MB的文件传输可能较慢
3. **应用图标**：使用默认Electron图标
